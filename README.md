- **Cinematic Opening**: Black screen, piano music, stars forming, heart animation

### Interactive Elements
- **40+ Emotional Questions**: Deep, meaningful questions about love and relationships
- **Choice-Based Narrative**: Every answer influences the emotional journey

- **Background Music**: Cinematic piano soundtrack

## 🚀 Getting Started

### Installation

1. Clone this repository:
```bash
git clone https://github.com/godfreymuthomi23-netizen/love-story.git
cd love-story
```

2. Open `index.html` in a modern web browser:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Or simply open in browser
open index.html
```

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No external dependencies required
- Uses only HTML, CSS, and JavaScript
- Google Fonts for typography

## 🔧 Customization

### Change Questions
Edit the `QUESTIONS` array in `script.js`:
```javascript
const QUESTIONS = [
    {
        text: "Your question here?",
        choices: ["Choice 1", "Choice 2", "Choice 3"],
        quote: "Relevant quote here"
    },
    // ... more questions
];
```

### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-pink: #ff1744;
    --crimson: #c41e3a;
    --gold: #ffd700;
    /* ... more variables */
}
```

### Change Text
Edit the opening and ending text in `index.html`:
- Opening scene messages
- Ending declaration
- Button labels
